# How to add a Vivado project to Git.
## directory tree
```
repo/
  src/        (HDL,copy of <project_name>.srcs that is generated by vivado)
  constr/     (XDC)
  bd/         (BD,bd.tcl)
  ip/         (custom IP,xci)
  scripts/
    <project_name>.tcl
    build.tcl
```
## generating files.
### How to generate <project_name>.tcl.
1. Vivado Menu. [File]->[Project]->[Write Tcl...]. 
1. Open "Write Project to Tcl" Dialog.
1. Check "Copy sources to new project" and "Recreate Block Designs using Tcl".
1. select a output file. `repo/scripts/<project_name>.tcl`
1. Push [OK].
### Edit <project_name>.tcl
1. copy <project_name>.srcs to `repo/src`
1. `repo/scripts/<project_name>.tcl` 
1. Edit <project_name>.tcl so that, during project reconstruction, it selects the required files from repo/{src,constr,bd,ip}/.

## How to recreate a project.
`vivado -mode batch -source scripts/<project_name>.tcl`
